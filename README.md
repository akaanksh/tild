# Tild CLI
Get it from npm:
```bash
$ npm i tild
```

### Generate Files
```bash
$ tild g m
```

### Open Links
```bash
# Opens Tild's documentation:
$ tild o d
# Opens Tild's Website:
$ tild o s
# Opens Tild Minimal's Landing Page:
$ tild o m
# Opens Tild Pro's Landing Page:
$ tild o p
# Opens Tild's CDN Links Page:
$ tild o c
```
